Use maven in order to run this project.
This project runs with an embedded tomcat provided by springboot and a H2 in memory database for runtime and also for testing purposes.
The endpoints are in BookingController and CampsiteController classes.


About the model 
The campsite is organized in spots, so the user can do reservations of spots. 
The model allows multiple campsites(for future rest API modifications), so far VOLCANO campsite is the only one available.
